# Agent Skills 生态

## 定义

Agent Skills 是一种将指令、脚本和资源打包为可复用模块的模式，让 AI 编码 Agent 能够以结构化、可重复的方式完成特定任务。2026 年 5 月，这一模式在 GitHub Trending 中集中爆发。

## 今日上榜的 Agent Skills 项目（5/12）

- [[obra-superpowers]] — 193K star，最成熟的 agentic skills 框架，强制 TDD 和结构化工作流
- [[anthropics-skills]] — 135K star，Anthropic 官方技能库，定义了 Agent Skills 标准
- [[mattpocock-skills]] — 85K star，聚焦工程实践四大失败模式（对齐、冗余、不工作、设计腐烂）
- [[K-Dense-AI-scientific-agent-skills]] — 22K star，135 个科研专用技能，覆盖 78+ 科学数据库
- [[czlonkowski-n8n-mcp]] — 20K star，MCP 桥接 n8n 工作流自动化

## 趋势分析

**为何集中爆发？**
- Agent Skills 标准（agentskills.io）趋于成熟，降低了技能创建和分发的门槛
- AI 编码 Agent（Claude Code、Codex、Cursor）用户量激增，对技能的需求暴涨
- 从通用到垂直：生态从通用开发技能扩展到科研、工作流自动化等垂直领域

**关键信号：**
- 5 个项目合计 456K+ star，占今日上榜总 star 的显著比例
- Anthropic、社区个人开发者（mattpocock、obra）和垂直领域公司（K-Dense）同时活跃
- MCP 协议成为 Agent 与外部工具交互的标准桥梁
