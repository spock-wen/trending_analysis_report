# anthropics/skills

## 概述

Anthropic 官方的 Agent Skills 公开仓库。Skills 是由指令、脚本和资源组成的文件夹，Claude 动态加载以提升特定任务表现。包含创意设计、开发技术、企业通信、文档处理等多个类别的示例技能。135K star。

## 核心功能

- **Agent Skills 标准** — 定义技能规范（SKILL.md + YAML frontmatter）
- **多类别技能** — 创意设计、开发技术、企业工作流、文档处理
- **文档技能** — DOCX、PDF、PPTX、XLSX 创建和编辑
- **跨平台使用** — Claude Code（插件市场）、Claude.ai（付费版）、Claude API
- **技能模板** — 提供创建自定义技能的模板和指南

## 技术栈

- Python（84.4%）+ HTML（12.4%）
- Agent Skills 规范（agentskills.io）
- Claude 插件系统

## 使用场景

- Claude 用户：扩展 Claude 的专业领域能力
- 企业：创建品牌指南、数据工作流等定制技能
- 开发者：学习 Agent Skills 标准和最佳实践
