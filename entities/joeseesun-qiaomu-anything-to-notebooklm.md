# joeseesun/qiaomu-anything-to-notebooklm

## 概述

Claude Code Skill，将 15+ 种来源内容（含付费内容）自动处理并上传至 Google NotebookLM，生成播客、PPT、思维导图、Quiz 等多种格式。支持 300+ 付费网站的付费墙绕过。

## 核心功能

- **15+ 内容源** — 微信公众号、X/Twitter、YouTube、播客、PDF、EPUB、Office 文档等
- **6 层付费墙绕过** — 代理服务 → 站点专属 Bot UA → 通用绕过 → archive.today → Google Cache → 本地工具
- **300+ 付费网站** — NYT、WSJ、FT、Economist、Bloomberg、The Information 等
- **8 种输出格式** — 播客、PPT、思维导图、Quiz、视频、报告、信息图、闪卡
- **深度分析模式** — 三轮递进式分析：概览框架 → 深度挖掘 → 综合反刍

- 将 YouTube 视频转为思维导图
- 批量处理多种格式内容并统一输出
