# oven-sh/bun

## 概述

极速 JavaScript/TypeScript 一体化工具包：运行时 + 包管理器 + 测试运行器 + 打包器。用 Zig 编写，底层基于 JavaScriptCore 引擎，作为 Node.js 的直接替代品。90K star，15,360+ commits。

## 核心功能

- **Bun 运行时** — Node.js 直接替代品，启动速度和内存使用大幅优化
- **包管理器** — `bun install`/`add`/`remove`/`update`，兼容 Node.js 生态
- **测试运行器** — 内置 `bun test` 测试框架
- **打包器** — 内置 `Bun.build` 打包工具
- **丰富 API** — Bun.serve、Bun.sqlite、Bun.redis、Bun.s3、Bun.cron、Bun.YAML 等
