### 本周总览

本周 GitHub Trending 周榜共有 18 个项目上榜，累计获得 107,082 颗星标，平均每个项目约 5,949 stars，热度处于近几周高位。从语言分布来看，TypeScript 以 9 个项目（占 50%）成为绝对主力，Python 以 5 个项目（占 28%）紧随其后，Rust 有 2 个项目（11%），Swift 和 Shell 各有 1 个。对比上周的日榜数据，TypeScript 占比从约 27% 飙升至 50%，Python 则从 43% 降至 28%，反映出 AI 开发工具链正在经历从 Python 主导向 TypeScript/JavaScript 生态的加速迁移。本周新上榜项目 8 个，爆发型项目 3 个（单日峰值超过 1,800 星），二次爆发项目 5 个，市场情绪极为活跃。

- **tinyhumansai/openhuman**：Rust 实现的个人 AI 超脑，自 5 月 16 日首次上榜以来累计 23,651 星，本周再获 15,194 星排名第 2。连续上榜 4 天、最高排名 #1 的记录表明，Rust 在 AI 推理基础设施层的渗透正在加速。值得注意的是，openhuman 主打"私有、简单、强大"，与云端大模型形成差异化竞争。

- **Imbad0202/academic-research-skills**：为 Claude Code 提供端到端学术研究流程（研究、写作、审稿、修改、定稿），本周获 11,401 星。它标志着 AI Agent 与科研工作流的深度融合不再是概念，而是已经产生真实下载量的成熟产品。

核心判断：以 Claude Code 为载体的 Agent Skills 生态正在从"单一工具 wrapper"走向"全链条研发工作流解决方案"。这不是短期炒作，而是 AI 飞轮效应的开端。

### 领域热度对比

对比上周（5/18-5/24）和本周（5/25-5/31）的日榜数据，各领域热度变化如下：

- **AI 开发工具与 Agent 框架**：保持最热赛道，但内部格局在变。上周以 codegraph 和 openhuman 为主，本周 codegraph 以 18,136 星登顶周榜，Understand-Anything（9,102 星）和 oh-my-pi（2,361 星）涌现。Agent 工具链正在从"单一功能"向"全栈平台"快速演进。

- **Claude Code Skills 生态**：上周以 academic-research-skills 和 claude-plugins-official 为代表，本周进一步扩展出 knowledge-work-plugins 和 Anthropic-Cybersecurity-Skills 等垂直技能包。Anthropic 正在构建类似 VS Code 扩展市场的 Agent Skills 应用商店生态。

- **Python 项目**：日榜数量从上周 18 个降至本周 7 个，但在周榜中仍有 5 个项目（占 28%）。Python 在 AI 应用层（CLI-Anything、Academic Research Skills）和工具层（CloakBrowser）的地位依然稳固，但增长动能正在减弱。

- **TypeScript 项目**：在周榜中占 50%，取代 Python 成为本周最活跃语言。Cursor、Claude Code 等产品以 TypeScript 为第一公民语言，带动了整个 Agent 工具链的 TypeScript 化。这可能是 AI 开发生态的一个长期趋势转折点。

- **安全与浏览器自动化**：CloakBrowser 以 6,892 星排名第 8，Chrome DevTools MCP 以 1,732 星排名第 18。浏览器自动化与反检测需求持续走强，AI Agent 与浏览器的交互正在成为新的核心技术栈。

### 下周关注

基于本周数据和多维信号，我给出以下五项预测性判断：

1. **CodeGraph 的卫冕之战**：codegraph 本周以近 2 倍于第 2 名的成绩夺冠，但代码知识图谱赛道正在快速拥挤化（Understand-Anything、oh-my-pi 等均在相近方向）。codegraph 能否持续领先，取决于其技术深度的差异化——预索引的质量、代码覆盖率、多 IDE 兼容性等。

2. **Agent Skills 生态的横向爆发**：从 academic-research 到 cybersecurity 再到 knowledge-work，Anthropic 的 Skills 体系正在快速横向铺开。下周很可能看到更多垂直领域技能包（如法律、医疗、金融）进入趋势榜。这将是一个持续数月的长周期趋势。

3. **TypeScript 主导地位的确认**：本周 TypeScript 占比 50% 是近几个月罕见的数字。如果下周趋势延续，意味着 AI Agent 工具链正在完成从 Python 到 TypeScript 的范式切换。所有 AI 开发工具链的创业者应认真对待这一信号。

4. **浏览器自动化赛道的升温**：CloakBrowser（6,892 星）加上 Chrome DevTools MCP（1,732 星）正在形成合力。AI Agent 需要强大的浏览器交互能力来执行 Web 端任务，这个细分赛道的战略价值被低估了。

5. **RuView 代表的跨域实验**：用 WiFi 信号做空间智能和生命体征监测的 Rust 项目，虽然只有 6,461 星，但它的跨界思路值得高度关注。当 AI 从纯代码世界走向物理世界，这类传感器与 AI 结合的项目可能成为下一个爆发的起点。
